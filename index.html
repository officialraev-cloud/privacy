Политика конфиденциальности

Приложение «Цифры» (далее — Приложение) уделяет большое внимание безопасности ваших данных. Данная политика описывает, как мы обрабатываем информацию.

1. Сбор информации и Идентификаторы

При использовании функции «Вход с Apple» Приложение получает ваш уникальный идентификатор пользователя (UID) и публичное имя профиля. Мы также можем собирать анонимные идентификаторы устройств для аналитики и улучшения стабильности работы (диагностики сбоев).

2. Финансовые данные и Синхронизация

Ваши записи о транзакциях, бюджетах, целях и активах хранятся в двух контурах:

  • iCloud: для обеспечения приватного хранения данных в экосистеме Apple.
  • Firebase (Google Cloud): для обеспечения синхронизации данных между вашими устройствами и кроссплатформенного доступа.

Финансовые данные хранятся в облачной базе в зашифрованном бинарном виде. Разработчик Приложения не использует эту информацию для идентификации вашей личности в реальном мире и не передает её третьим лицам.

3. Сторонние сервисы

Для обеспечения работы функций мы используем следующие сервисы:

  • Firebase (Google): для авторизации, хранения профилей и облачной синхронизации данных.
  • Apple Speech Recognition: для обработки голосовых команд.
  • GigaChat (Sber): для лингвистического анализа текста операций (передается только текст фразы без привязки к личности пользователя).

4. Удаление данных

Вы имеете полный контроль над своей информацией. Вы можете в любой момент полностью удалить все свои данные (включая профиль, историю операций и активы), используя функцию «Сброс всех данных» в настройках Приложения. Это действие безвозвратно удалит информацию с устройства, из iCloud и из облачной базы Firebase.
